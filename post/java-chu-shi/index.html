Java初识

2020-04-30 1 min read # Java

1.Java凭什么这么强大？
主要由JVM实现平台无关性，真正实现一处编译，随处运行。

2.JVM、JRE及JDK的区别？
JVM：Java虚拟机
JRE：JVM + Java SE标准类库
JDK：JRE + 开发工具（例如Javac编译工具等）

3.Java分类
Java SE（标准版）
Java EE（企业版）
Java ME（手机版）

4.一个Java程序的开发流程（粗略）
1）编写源代码 .java
2） javac.exe编译源代码 ，生成字节码文件
3） java.exe运行字节码
